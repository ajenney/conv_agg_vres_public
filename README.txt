Code used to create the figures in the manuscript,
"Vertical resolution impacts explicit simulation of deep convection"
submitted to JAMES by A. M. Jenney, S. L. Ferretti, & M. S. Pritchard (2022).

The code uses output from simulations with the System for Atmospheric
Modeling (SAM) version 6.10.9.
